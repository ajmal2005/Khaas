Khaas is a dual-purpose utility app designed for personal safety and social management. It overrides system "Silent/Do Not Disturb" modes for specific "VIP" contacts, ensuring you wake up or get alerted for emergencies. Conversely, it features a "Fake Call" escape button to gracefully exit unwanted situations.
